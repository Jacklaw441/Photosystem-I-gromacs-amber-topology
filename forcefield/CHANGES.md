# Force Field Changes

This file documents all modifications made to the stock AMBER03 force 
field as distributed with GROMACS 2021. All changes were made to 
accommodate the non-standard residues and cofactors present in 
Photosystem I (PSI).

## posre_ffbonded.itp and posre_ffnonbonded.itp

These files were updated to reflect the new bonded and non-bonded 
parameters described above. No novel parameters were introduced 
beyond those already documented in the ffbonded.itp and 
ffnonbonded.itp sections. These files are used only during the 
NVT and NPT equilibration stages and are not active during 
production MD.
